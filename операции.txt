Комбинации клавиш------------------------------------------------------------------
ctrl + E - В конец строки
ctrl + А - В начало строки
alt + F - Перемещение вправо по пробелам
alt + B - Перемещение влево по пробелам
ctrl + K - Удалить строку c начала строки
ctrl + U - Удалить строку с какого-то момента
ctrl + Y - Вернуть строку
ctrl + D - Удалит с символа
alt + U - Вернхний регистор для всех символов слова (курсор в начале слова)
alt + C - Слово с большой буквы
alt + L - Нижний регистр для всех букв слова
ctrl + T - Меняет местами 2 последние символа слова
alt + T - Меняет местави 2 последних слова
ctrl + shift + T - новая вкладка в терминале

Основные команды терминала----------------------------------------------------------
"ls" (list - список) - покажет, что находится в текущем каталоге
	"ls -a" - покажет и скрытые файлы
	"ls -l" - покажет полную информацию (например, права, дату и тд)
	"ls -R" - покажет файлы до конца каталогов (рекурсивно)
	"ls -m" - покажет файлы через запятую в текущем каталоге
	"ls -F" - покажет информацию о типах файлов
	"ls -r" (reverse) - покажет файлы в обратном порядке
	"ls -x *.txt" - сортировка файлов вместе .txt
	"ls *.txt" - покажет только файлы .txt (маска)
	"ls --color" - покажет файлы в разных цветах
"pwd" (print working directory) - покажет, в каком каталоге находимся
"cd" (change directory - сменить директорию) - переместиться в выбранную папку
	"cd ~" - Самая главная папка
	"cd .." подняться на уровень выше
	"cd -" переместиться в предыдущий каталог
"touch 1 2 3 4" создаст пустые файлы "1", "2", "3", "4"
	"touch Музыка" - изменит время обращения к папке "Музыка" на текущее время
	"touch -t 01021530 Dairy" - изменит время обращения папки Dairy на 2 января 15:30
"mkdir" (Make directory) - создать папку
	"mkdir -p papka1/papka2/papka3/papka4"/ создаст рекурсию. папка 4 в папке 3, та в папке 2, и всё это в папке 1
"cp text1 text2" - скопирует файл 1 в файл 2, перезаписав файл 2!
	"cp *.txt primer1/" скопирует все фалйы .txt в папку primer1
	"cp 1[7-8].txt primer1" скопирует файлы по маске 17.txt 18.txt в папку primer1
	"cp -v one six" отобразит изменения в терминале: 'one' -> 'six'
	"cp -i text1 text2" - спросит, хотим ли мы перезаписать файл text2
	"cp -R papka1 primer" скопирует рекурсивно все файлы papka1 в папку primer
"mv two primer1/" (move) - переместит файл two в папку пример1
	"mv one seven" переименует файл one в seven 
	"mv -i one two" спросит, хотим ли переписать файл two
"rm 19.txt" (remove) удалит файл 19.txt
	"rm *.text" удалит все файлы .text
	"rm -v six" (verbose - многословный) - отобразит в терминале: удалён 'six'
	"rm - R" (recursive) - рекурсивно удалит файлы
	"rm - f" (forced) - удалит без вопросов и ошибок
	"rm - i" (interactive) - спросит, хотим ли мы удалить файл
	"rmdir" удалит пустую папку
"man ls" (manual) - подробная информация по команде ls
"whereis ls" - покажет, где находятся части команды ls
"whereis -b gimp" - покажет директорию исполняемого файла gimp
"which ls" - покажет директорию исполняемого файла ls, стоит ли данная программа в системе
"whatis ls" - описание команды ls (полезно)
Последовательности команд и значения символов:
	a ; b - последовательно выполнит команды а, потом б
	a && b - выполнит команду б в случае успешного выполнения команды а
	a || b -  выполнит команду б в случае НЕ выполнения команды а
	a | b - (канал) передаст выходные данные с а на вход б
	a > b.txt - сохранит результат команды а в файл b.txt
		d > b.txt - перезапишет результат b.txt  в тот же файл
		f >> b.txt - дозапишет выходные команды f в тот же файл
	echo < b.txt - выгрузит в команду echo файл b.txt
"cat text.txt" - покажет, что находится внутри файла text.txt
	"cat 1.txt 2.txt > 3.txt" загрузит в файл 3.txt, склеив 1.txt и 2.txt
"less 1.txt" - откроет большой текстовый файл в интерфейсе с удобным перемещением
"head 1.txt" - покажет первые 10 строк файла
	"head 1.txt 2.txt" - покаже 10 строк 1 и 2 файла
	"head -n 7 1.txt" - выведет первые 7 строк файла
	"head -c 100 1.txt" - выведет первые 100 байт информации файла
"tail 1.txt" - покажет последние 10 строк файла. Для этой команды такие же ключи, как и для head
"df -h" - покажет размеры файлов
"free -h" - покажет загруженность оперативной памяти
"chmod 777 test" - даст права на чтение, запись, исполнение для user, group, other (7 - u, 7 - g, 7 - o)
"zip 1.zip 1.txt" - архивирует 1.txt в zip, не удаляя файл 1.txt
	"unzip 1.zip" - разархивирует zip
	"zip -e test.zip 1.txt 2.txt 3.txt 4.txt" - архивирует 4 файла, ключ -e - вызовет установку пароля на этот архив 
	"zip -d test.zip" -расжатие файла
	"tar cvf 1.tar test" - заархивирует тест в 1.tar, ключ v покаже, что происходит
	"tar xf 1.tar" - разархивирует tar.1
	"gzip 1.tar" скомпрессирует архив 1.tar
	"gunzip 1.tar.gz" раскомпрессирует архив
"locate Panzar" - обратится к бд, если найдёт Panzar - покажет путь
	"locate -i panzar" - поиск без учёта регистра
"grep panzar" - выведет строку, в которой есть "panzar"
	"grep -Ri panzar *" поиск в подкаталогах (R) без учета регистра (i) искать везде (*)
	"grep -wR panzar *" - рекурсивно искать везде исключительно слово panzar без вхождений в другие слова
	"grep -n panzar" - выведет номер строки и саму строку с ключевым словом
"find -size 10k" найти файлы по размеру 10 кб
"apt update" - список пакетов
	"apt upgrade" - установка пакетов
	"apt remove 1" - удаление пакета 1
	"apt install 1" - установление пакета 1
	"apt dist-upgrade" - полностью обновит дистрибутив 
	"apt clean" - очистит остатки
	"apt -f install" - если нарушены зависимости
	"wget" (web get) - скачать файл из интернета, указав путь
"ln -s путь1 путь2" - создаст ярылк (шорткат линк) файла пути 1 в путь 2
"wc 1.txt" (word count) - выдаст кол-во строк, кол-во строк и кол-во байт
"sort 1.txt" - сортировка файла 1.txt по группам ааа ввв ддд оо... 999 000 (не алфавит и не по возраст)
	"sort -n 1.txt" - сортировка по возрастанию 
"cut -d ">" -f 3 1.txt" - вырежет все нужное по разделителю (-d == >), после 3 разделения (-f) из файла
"su" - сменить текущего пользователя
	"whoami" - покажет имя текущего пользователя
	"id" - выдает группы пользователя
	"who" - покажет кто в системе сейчас
	"w" - покажет кто в системе и что делает
	"last" - покажет последний логин в систему
"ifconfig" - выдаст информацию про сеть (айпи и тд)
	"route" - информация про роутер 
	"ping 8.8.8.8" -  проверить соединение с гуглом
	"traceroute 8.8.8.8" - покажет узлы прохждения пакетов до гугла
	"host google.com" - покажет айпи гугла
	"netstat" - сетевые подключения компьютера
	"гfw allow 22" - откроет порт 22

	"git pull origin master" - запрос изменения с сервера
	"git branch First" - создание новой ветки

	"git merge First" - слияние веток First и мастер
	"git branch -d First" - удаляем ветку First, поскольку она не нужна больше
	"git log" - лог изменений
	"git show 111111" - покажет инфу про коммит
	"git diff 111111..222222" -покажет разницу между коммитом 1 и 2
